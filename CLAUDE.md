# Portfolio Project Instructions

## Overview
Personal portfolio site for Amanda Low, Product Designer. Minimal, editorial aesthetic inspired by Squarespace Aue Sobol template.

## Typography Rules

### Fonts (Google Fonts)
- **Averia Serif Libre Bold** - All headings and titles, ALWAYS UPPERCASE
- **Manrope Light (300)** - Body text, navigation, descriptions

### Key Decisions
- Homepage description uses Manrope Light, sentence case (NOT uppercase)
- All Averia Serif Libre text must be uppercase
- Navigation uses Manrope Light uppercase with letter-spacing 0.08em
- Active nav link is underlined (not dimmed)

## Color Palette
- Background: `#FAFAF8` (warm off-white)
- Text: `#1A1A1A`
- Muted text: `#737373`
- Border: `#E8E8E6`

## Layout Constraints
- Max content width: 1200px
- Site padding: 3rem (desktop), 1.5rem (mobile)
- All content areas must have consistent width and padding

## Page Structures

### Homepage
- Centered header with logo + nav stacked
- Hero description (Manrope Light, left-aligned, sentence case)
- Work grid with alternating image positions

### About Page
- Two-column: title + bio on left, photo on right
- Photo top aligns with "ABOUT" title (use margin-top on image)
- Photo displays uncropped at natural aspect ratio

### Contact Page
- Two-column: title + details on left, form on right
- Form has First Name/Last Name side by side
- Input borders are 2px solid black

## Responsive
- Breakpoint: 768px
- Mobile: single column layouts, reduced padding

## Files
- `css/styles.css` - All styles with CSS variables
- `js/main.js` - Scroll animations, form handling
- See `STYLEGUIDE.md` for complete documentation
