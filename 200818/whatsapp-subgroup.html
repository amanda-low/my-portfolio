Problem

Group chats are information overload.

Users can miss out on information that is relevant to them because they mute or ignore large group chats.

Solution

Design a new feature: Subgrouping chats by topic


Method
Method
Define User Flow

The new feature should be integrated as to not interrupt the existing user flow. It should be simple, to cater to the wide user demographic, and intuitive.

I tested and researched the current features related to the group chat to find ways to add seamlessly to the flow.
Rapid Prototyping

To assess the usability of possible solutions, I user tested with lo-fi prototypes focusing on what users found intuitive, or not.
